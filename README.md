# E-COMMERCE-APP: FIVE SALADS
Responsive web application for food purchase and delivery with an option for account registration. 

## Build using:
- HTML
- CSS
- JavaScript
- Java

- Maven
- Spring Boot + Spring Security
- Thymeleaf
- H2 Database
